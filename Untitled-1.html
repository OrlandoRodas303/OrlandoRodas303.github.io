Introducción

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar utilizado para crear y diseñar páginas web. Define la estructura de una página mediante el uso de elementos representados por etiquetas. Estas etiquetas le indican al navegador cómo mostrar el contenido como texto, imágenes, enlaces y más.

Elementos Clave

Existen varios elementos clave en HTML que definen la estructura de una página web:

  • <!DOCTYPE html>: Define el tipo de documento y la versión de HTML.
  • <html>: El elemento raíz de la página HTML.
  • <head>: Contiene metadatos como el título y enlaces a estilos CSS.
  • <title>: Define el título que aparece en la pestaña del navegador.
  • <body>: Contiene el contenido visible de la página (texto, imágenes, etc.).
  • <h1> a <h6>: Etiquetas de encabezado para títulos (del mayor al menor).
